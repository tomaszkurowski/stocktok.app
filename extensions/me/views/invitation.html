Safari & Facebook browser note

There are two technology novelties inside app:

  • PWA-test
  • Webauthn (device authorization with fingerprint, faceID etc.

They should be working fine in Chrome/Firefox/Edge browsers.

Safarii unfortunatelly is still working on them and they are implemented only partially. PWA will not have button "add to homescreen" (so it can be added only manually). Webauthn works only with external (USB) devices.

FB browser is not supporting both of them at all.
Therefor, if you want best experience, please open app in Chrome.
