IOS Safari note

There are two technology trend-features inside app:

  • PWA
  • Webauthn (Device authorization with fingerprint, faceID, PIN)

They should be working fine in Chrome/Firefox/Edge browsers.

Safarii unfortunatelly is still working on them and they are implemented only partially. PWA will not have button "add to homescreen" (so you can add app only manually). Webauthn works only with external (USB) devices.
